# 🌦️ Meteorological Data Analysis Web Application

## 📌 Overview
The **Meteorological Data Analysis Web Application** is an interactive platform designed to provide users with in-depth analysis of meteorological datasets.  
It combines **data processing**, **visualization**, and **prediction models** to deliver insights about temperature trends, rainfall patterns, humidity levels, and more — all in a clean, user-friendly interface.

---

## ✨ Key Features
- 📊 **Interactive Data Visualization** – View meteorological trends with intuitive charts and graphs.
- 🔍 **Search & Filter** – Quickly filter datasets based on time range, location, or parameters.
- 📈 **Statistical Insights** – Get min, max, average, and trend line analysis for multiple parameters.
- 🤖 **Machine Learning Predictions** – Forecast temperature and rainfall based on historical data.
- 🖥️ **Clean UI** – Designed with accessibility and usability in mind.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** Python Flask
- **Data Analysis:** Pandas, NumPy, Matplotlib
- **Machine Learning:** Scikit-learn
- **Deployment:** Render (optional for live access)

## ⚙️ Installation & Usage

1️⃣ Clone the Repository
git clone https://github.com/yourusername/meteorology-project.git
cd meteorology-project

2️⃣ Create & Activate Virtual Environment
python -m venv venv
source venv/bin/activate   # For Mac/Linux
venv\Scripts\activate      # For Windows

3️⃣ Install Dependencies
pip install -r requirements.txt

4️⃣ Run the Application
python app.py

The application will run on:
http://127.0.0.1:5000
